# Tempo Documentation

Welcome to the Tempo documentation. Tempo is a full-stack team productivity platform built with Next.js 15, MongoDB, and NextAuth.js. It combines task management, projects, chat, calendar, video calls, focus sessions, and real-time notifications into a single workspace.

## Quick Overview

| Feature | Description |
|---|---|
| **Tasks** | Kanban board with drag-and-drop, priorities, due dates, project tags, and assignees |
| **Projects** | Group tasks under named projects with progress tracking |
| **Calendar** | Schedule events, view by month/week/day, shared team calendar |
| **Chat** | Real-time messaging via polling across team channels |
| **Pings** | Instant lightweight notifications to teammates |
| **Notes** | Personal markdown-style notes |
| **Focus** | Pomodoro-style focus session timer |
| **Calls** | WebRTC peer-to-peer video rooms |
| **Notifications** | In-app toast alerts + browser desktop notifications |
| **Presence** | Live online/offline status for all team members |
